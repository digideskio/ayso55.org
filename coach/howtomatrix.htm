How to use the Matrix

Consider each player against the matrix only. Do not consider him against other players. A common question arises at this point which is "what do I give a player who is clearly the best? or "what do I rate a player who barely knows there is a soccer match going on around him?" The answer is simply whatever the matrix best dictates, from each column. If not done this way, the problems arise when considering the middle of the road players; you begin to compare them to your best or worst, rating then become subjective or bias and not objective based strictly on the matrix. Bottom line....use the matrix for each player, the best player will receive the highest rating and the worst the lowest.

 Starting with the first column, determine the description that best fits the player you are rating, list the number that corresponds to the description. Continue moving to the next column and listing the number, until all columns are complete. Then simply ADD all columns and you have the total rating for that player. Do not average the columns, add them together.

 Should a player fall between two descriptions use the lower number since the player does not fully meet the description the corresponds to the higher number. As an example Johnny/Jenny is definitely a 3 but does not quite fit the description of a 4, Johnny would receive a 3 for that column.

 Note that if your matrix has shaded descriptions, these are not to be considered or used for your age group. The matrix is used for all age groups and is tailored for a particular group by shading descriptions and corresponding numbers that are not applicable. An example would be U10 balls skills and speed column where the highest number a player could receive for this column is 5 since the description for 6 is shaded.
